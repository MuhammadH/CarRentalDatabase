This was a school project for creating a car rental database system using sql/ php/ html. 

Install XAMPP

Run XAMPP. 
Start MySQL, ProFTPD, and Apache services. 
Enable localhost:8080 -> 80 (Over SSH) in Network
Mount /opt/lampp

Import the database (.sql) CAR_RENTALS_HUSSAIN into phpMyAdmin

Put CAR_RENTALS in lampp/htdocs

Go to
http://localhost:8080/CAR_RENTALS/
In a web browser

And have fun! The full database is on the bottom of the main (index) page
